📊 Job Market Data Analyzer

📌 Overview



This project analyzes job market trends using real-world job salary data. It demonstrates data cleaning, analysis, visualization, and SQL integration using Python.



🛠️ Tech Stack

Python (pandas, matplotlib)

SQL (SQLite)

📊 Features

Data cleaning and preprocessing

Analysis of:

Top job roles

Salary by experience level

Remote work distribution

Top company locations

Data stored in SQL database

Visualizations using charts

▶️ How to Run

pip install -r requirements.txt

cd scripts

python data\_cleaning.py

python analysis.py

python db\_setup.py

📈 Key Insights

Data Scientist and Data Engineer are the most common roles

Executive-level salaries are significantly higher

Majority of jobs are fully remote

The US leads in job opportunities
